# noteapp

### A Note pad App to Add, Save Notes, Pictures and Audios

## Features
 - Authentications
 - 2 factor OTP authentication
 - Adding, Deleting & Editing notebooks, notes and chatrooms with labels
 - Animations in notes
 - adding picture, messages and audio feature
 - notes layouts
